# Viajes E-commerce

Proyecto final del curso React JS - Coderhouse

## Tecnologías
- React
- React Router
- Firebase (Firestore)
- Context API
- Bootstrap

- Carrito de compras
- Formulario de compra con generación de orden en Firebase
